# CLI Inventory Management Tool
A Java-based command-line inventory management system that allows users to add, update, delete, and view inventory items.  
This project demonstrates object-oriented programming, data structures, and input validation.

## Features
- Add new inventory items
- Update item quantities
- Remove items
- View complete inventory list
- Input validation and menu-driven interface

## Technologies
- Java
- Object-Oriented Programming (OOP)
- Command-Line Interface (CLI)

## How to Run
1. Compile the program:
   javac InventoryManager.java

2. Run the program:
   java InventoryManager

## You will see

1. Add Item
2. View Inventory
3. Update Quantity
4. Remove Item
5. Exit
